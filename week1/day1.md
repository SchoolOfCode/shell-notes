## CSS

### 3 ways to apply styles

```html
<!-- inline styles -->
<h1 style="font-size: 16px;">Title</h1>

<!-- using the style tag -->
<style>
  body {
    background-color: white;
  }
  h1 {
    color: #00ff00;
  }
</style>

<!-- linking a separate style sheet -->
<link rel="stylesheet" type="text/css" href="myStyles.css" />
```

### CSS selectors

```css
tagName {
  /* Styles go here */
}

.className {
  /* Styles go here */
}

#idName {
  /* Styles go here */
}
```
